以 IntelliJ IDEA 入门

最近更新 2020-01-23
本教程将引导你使用 IntelliJ IDEA 创建一个简单的 Hello World 应用程序。

搭建环境

在本教程中将使用 IntelliJ IDEA。首先，安装一个最新版本的 IntelliJ IDEA。

从 15 版开始，IntelliJ IDEA 内集成 Kotlin。

可以从 JetBrains 官网下载免费的社区版（或完整的旗舰版）。

作为使用 IntelliJ IDEA 的替代方法，也可使用命令行编译器来编译和执行 Kotlin 应用程序。相关详细信息，请参见使用命令行编译器。

如果不熟悉 JVM 和 Java，请查看 JVM 简明使用手册。 如果您不熟悉 IntelliJ IDEA，请参见 IntelliJ IDEA 简明使用手册。

创建一个新的项目

安装好 IntelliJ IDEA 之后，就可以创建第一个 Kotlin 应用程序了。

  1. 从 File | New 开始（如果这是你第一次使用 IntelliJ IDEA，从 Create New Project 开始）创建一个新的项目。选择 Kotlin | JVM | IDEA 作为项目的类型。

  2. 为项目命名并为其选择 SDK 版本。

    现在，你已经创建了一个新的项目。以下是项目结构：

  3. 在 src 文件夹下创建一个新的 Kotlin 文件。 它可以任意命名。 我们称之为 app。

  4. 创建文件后，添加 main 函数，它是 Kotlin 应用程序的入口。 IntelliJ IDEA 提供了一个快速完成此操作的模板。 只需输入 main 并按 Tab 键即可。

  5. 添加一行代码输出 “Hello，World！”。

运行这个应用程序

现在应用程序已准备好运行。 最简单的方法是单击菜单栏中的绿色 Run 图标，然后选择 Run'AppKt' 。

如果一切顺利，您将在 Run 工具窗口中看到结果。

恭喜！ 您现在正在运行第一个 Kotlin 应用程序。
